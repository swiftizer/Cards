Портрет пользователя: Студент
Студенты - это основная целевая аудитория нашего приложения. Они постоянно сталкиваются с необходимостью заучивания новой информации. Они ценят удобство и доступность, поэтому предпочитают электронные средства заучивания.

Сценарий использования №1: Создания набора карточек
 1. Пользователь открывает приложение.
 2. Пользователь нажимает "+" на главной странице
 3. Пользователь вводит наименование набора
 4. Пользователь нажимает "ADD"

Сценарий использования №2: Добавление карточки в набор
 1. Пользователь открывает приложение.
 2. Пользователь выбирает набор карточек
 3. Пользователь нажимает кнопку просмотра и редактирования карточек внизу экрана
 4. Пользователь нажимает "+" на главной странице
 5. Пользователь вводит вопрос, ответ, при желании дабавляет изображение
 6. Пользователь нажимает ">"

Сценарий использования №3: Заучивание карточки
 1. Пользователь открывает приложение.
 2. Пользователь выбирает набор карточек
 3. Пользователь просматривает вопрос карточки
 4. Пользователь нажимает на карточку
 5. Пользователь просматривает ответ карточки
 6. Пользователь кнопку с "галочкой" или "крестиком", помечая карточку как выученную/невыученную

Сценарий использования №4: Изменение настроек
 1. Пользователь открывает приложение.
 2. Пользователь нажимает на кнопку настроек
 3. Пользователь выставляет настройку "Is mixed" кнопкой в "YES"/"NO"
 4. Пользователь выставляет настройку "Mixing in power" ползунком в положение с нужной степенью подмешивания
